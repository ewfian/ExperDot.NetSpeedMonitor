##这是什么?
轻量级网络流量监控悬浮窗，左侧显示下载速度，右侧为上传速度。使用时手动设置为开机启动项即可。

##它有哪些功能？
* 网卡选择（默认自动选择Realtek PCIe GBE Family Controller ）
* 透明度选择
* 悬浮窗置顶
* 鼠标拖动，可自动停靠
